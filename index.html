Topic-1: Linux Intruduction

What is Linux?

Linux is an operating system just like Windows. But Windows is paid, while Linux is free and open-source. That is why many companies use Linux.

It was created by Linus Torvalds in 1991

* Difference Between Linux and Windows

  1. Multi-User Support
    Linux: Many users can connect at the same time using SSH.
    Windows: Only 1–2 remote users can connect. Extra users need paid licenses.
  2. Cost
    Linux: Free to use.
    Windows: Paid, requires license.
  3. Security
    Linux: Very secure, strong user permissions.
    Windows: More virus attacks, needs antivirus software.
  4. Speed
    Linux: Runs very fast, even on low RAM.
    Windows: Slower compared to Linux, because it uses heavy GUI.
  5. Interface
    Linux: command-based
    Windows: User‑friendly GUI
  6. Usage in Real-Time Projects
    Linux: Used in 95% of IT projects like Cloud, DevOps, Servers, Automation, Cybersecurity.
    Windows: Mainly used for personal work, offices, and gaming.

Linux Folder Structure

/home Users home directory
/root Root user home
/etc Configuration files
/var Logs, cache
/usr Applications & libraries
/opt Optional third‑party apps
/boot Bootloader files
/dev Device files
/proc Process info
/sys Kernel info
/tmp Temporary files

Important Ports Used in Linux Servers

Port 22 → SSH login  → Used to login to Linux servers securely.
Port 80 → Website     → Used for normal websites without security.
Port 443 → Secure website → SSL/TLS encrypted sites.
Port 3306 → MySQL database → Used by applications to connect to MySQL DB.
